1. Aceptación de los Términos

Al acceder y utilizar este sitio web, usted acepta estar sujeto a estos términos y condiciones de uso y acepta que es responsable del cumplimiento de todas las leyes locales aplicables. Si no está de acuerdo con alguno de estos términos, se le prohíbe acceder o utilizar este sitio.

Importante: El uso continuado del sitio web constituye la aceptación de estos términos.

2. Uso del Sitio Web

Uso Permitido

Este sitio web está destinado para:

  • Obtener información turística sobre Garzón, Huila
  • Consultar lugares de interés, eventos y servicios
  • Contactar con proveedores de servicios turísticos
  • Compartir experiencias y opiniones constructivas

Uso Prohibido

Está prohibido utilizar este sitio para:

  • Actividades ilegales o fraudulentas
  • Distribución de contenido ofensivo, difamatorio o discriminatorio
  • Ataques informáticos o intentos de comprometer la seguridad
  • Spam o distribución masiva de contenido no solicitado
  • Violación de derechos de propiedad intelectual

3. Contenido del Usuario

Al enviar contenido a nuestro sitio web (comentarios, valoraciones, fotos, etc.), usted garantiza que:

  • Es el propietario del contenido o tiene autorización para usarlo
  • El contenido no viola ningún derecho de terceros
  • El contenido es veraz y no engañoso
  • El contenido no contiene material ofensivo o inapropiado

Moderación: Nos reservamos el derecho de moderar, editar o eliminar cualquier contenido que no cumpla con estos términos.

4. Propiedad Intelectual

Todo el contenido de este sitio web, incluyendo pero no limitado a:

  • Textos, imágenes, videos y audio
  • Diseño gráfico y layout
  • Logotipos y marcas comerciales
  • Software y código fuente

Está protegido por las leyes de derechos de autor y propiedad intelectual de Colombia e internacionales.

Uso Autorizado

Se permite el uso personal y no comercial del contenido para fines informativos y turísticos, citando siempre la fuente.

  • No garantizamos la exactitud, actualidad o completitud de dicha información
  • No somos responsables de las políticas o prácticas de terceros
  • Las transacciones se realizan directamente con los proveedores
  • Recomendamos verificar la información antes de tomar decisiones
